Latihan CSS Background

1. Background Image & opacity

Section ini menggunakan gambar sebagai background

kontainer ini menggunakan opacity : 0.5, sehingga teks dan background container ini menjadi sedikit transparan terhadap gambar utama di belakangnya.

CSS: background-imageL url(....);
CSS: opacity: 0.85;

2. color, repeat, & size

Section ini menggabungkan background-color biru dengan gambar pola (pattern) transparan.

Pola tersebut diatur agar mengulang ke seluruh arah (repeat) dengan ukuran asli (size: auto).

kotak di atas mendemonstrasikan background-size : contain di dalam kontainer yang memiliki warna background berbeda.

CSS: background-color: #3498db;
CSS: background-repeat: repeat;
CSS: background-size: auto | contain | cover;

3. Background Position

Di sini, kita menggunakan gambar abstrack yang ukurannya di set manual (800px) dan no-repeat.

Gambar tersebut diposisikan secara spesifik di Kanan BawahSection ini.

CSS : background-position: right bottom;
CSS : background-repeat: no-repeat;

4. Background attachment (Fixed)

Ini adalah efek favorit banyak desainer: **Parallax**.

Coba scroll halaman ini naik turun. anda akan melihat bahwa gambar background di section ini diam (fixed) di tempatnya, sementara konten teks di atasnya bergerak melewati gambar.

CSS : background-attachment: fixed;
CSS : background-size: cover; (wajib agar menutup layar)
